# NumquamOblita

Evidence-backed memory for assistants that **doesn’t invent memories**.

- Local-first import → episodic memory build → review → safe runtime
- “Why this answer?” explainer with evidence + citations (toggleable)
- Built-in verifier that blocks unsupported recall (abstain/clarify)

If you want assistants that remember *real* things—and admit when they don’t—start here:
